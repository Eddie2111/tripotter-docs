# Project Introduction

Tripotter is a combination of apps, let's delve into this.

- Tripotter Web
- Tripotter Mobile
- Tripotter Pulse
- Tripotter Cappy
- Tripotter Market (Upcoming)
- Tripotter Beacon (Upcoming)

## TripOtter Web

Created with NextJS 16 App Router with Typescript and Turbopack. This project has all the social media features and required RestAPI's for social media features.

## TripOtter Mobile

Created with Expo-React Native with Typescript and Babel. Planned to release in all platforms via EAS.

## TripOtter Pulse

Create with NestJS. This project is the real time backend for TripOtter Web and TripOtter Mobile. This project also includes Graphql API's for history and trending related features

## TripOtter Cappy

Created with NestJS. This project has AI integration to combine search results and generate AI based responses to ease user searches.

## TripOtter Market

Proposed to create with Ruby on Rails. This project is the market place for TripOtter users to buy and sell their products.

## TripOtter Beacon

Proposed project to have a combining platform where any network and networkless communcation features can be implemented. Probable to use combination of AWS Amplify and AWS AppSync.
